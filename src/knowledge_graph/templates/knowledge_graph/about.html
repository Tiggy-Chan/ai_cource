关于《人工智能导论》课程知识图谱管理系统

欢迎使用《人工智能导论》课程知识图谱管理系统！

本系统旨在通过可视化的知识图谱、智能问答以及个性化学习笔记等功能，帮助用户更深入、高效地理解和学习《人工智能导论》课程中的核心概念、理论及其相互关系。

用户可以通过本系统进行以下操作：

  • 知识问答与图谱探索： 在首页输入感兴趣的知识点或问题，系统将展示相关的知识内容，并通过交互式图谱显示其在知识网络中的位置和联系。您可以点击图谱中的节点以探索更多关联知识。同时，系统还会展示与该知识点相关的公开学习笔记和您自己的相关笔记。
  • 知识总览： 通过"知识总览"页面，您可以系统地查看课程中所有已录入的知识点分类及其下的具体知识条目，方便地跳转到感兴趣的内容。
  • 学习笔记管理： 创建、编辑、查看和删除您的个人学习笔记。笔记支持富文本编辑、标签分类、设置公开或私有，并可以关联到具体的知识点。公开笔记可以通过唯一链接分享。
  • 智能问答助手： 在"智能助手"页面，您可以与集成通义千问的AI助手进行对话。助手可以回答您关于人工智能领域的问题，并且在回答时能够参考您已有的相关学习笔记，提供更个性化的解答。
  • 个人中心： 管理您的个人信息，包括编辑姓名、邮箱以及修改登录密码。

主要功能特色

  • 交互式知识图谱： 可视化展示知识点及其关联，支持点击节点动态加载邻居节点。
  • 智能检索： 基于关键词的知识点检索，支持多词AND逻辑及结果高亮。
  • 结构化浏览： 清晰的"知识总览"页面，方便用户按层级浏览知识体系。
  • 用户系统：完善的用户注册、登录、登出及个人资料管理（包括密码修改）。
  • 学习笔记：
    • 支持富文本编辑 (CKEditor)。
    • 标签化管理 (django-taggit)。
    • 公开/私有笔记及分享链接。
    • 笔记与知识点双向关联。
    • 笔记列表支持分页和按公开/私有筛选。
  • AI智能问答：
    • 集成阿里云通义千问大模型。
    • 支持流式输出，提升交互体验。
    • 能够结合用户的相关学习笔记进行个性化应答。
  • API支持： 提供知识库搜索API及图谱节点邻居查询API。

数据来源

本系统当前的知识数据主要来源于 人工智能导论-课程知识点.csv 文件。我们致力于不断完善和扩充知识库。

技术栈

本系统主要基于以下技术构建：

  • 后端： Python 3.x, Django Web框架 (包括 Django REST framework)
  • 前端： HTML5, CSS3, JavaScript (ES6+), Bootstrap (版本可能是4.x，需确认)
  • 知识图谱可视化： Vis.js
  • 富文本编辑器： CKEditor (django-ckeditor-uploader)
  • 标签系统： django-taggit
  • 大模型集成： 阿里云 DashScope SDK (通义千问)
  • 数据库： SQLite (Django默认，可轻松配置为 PostgreSQL, MySQL 等)
  • 版本控制： Git
